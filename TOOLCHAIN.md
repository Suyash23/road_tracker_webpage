# Toolchain Versions

To ensure stable and consistent builds (especially for iOS), we pin the following tool versions:

- **Flutter**: 3.22.0 (or newer 3.2x stable)
- **Xcode**: 15.0+
- **CocoaPods**: 1.15.0+

Always run `./scripts/doctor.sh` before building for iOS to ensure your local environment matches these requirements.
